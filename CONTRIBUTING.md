# Welcome contributors

Feel free to open up an issue if you have insights into bugs or even want to propose a change.

A non-exhaustive list of additions that are very welcomed are:

- OS support besides linux
- Ram based observations
- Porting more functions over from the original C++ <a href="https://github.com/openai/retro">Gym Retro</a> codebase
- Tooling in the form of reward tracking etc.
- Rewriting/Testing of existing features
- And many more
